Proyek Kalkulator BMI ini merupakan implementasi program menggunakan bahasa pemrograman Dart yang menerapkan kontrol alur program melalui percabangan untuk penentuan kategori berat badan dan perulangan untuk pengelolaan riwayat perhitungan. Seluruh proses pengembangan dilakukan dengan metodologi Git branching, di mana fitur-fitur utama seperti kalkulasi BMI, fitur riwayat, dan validasi input dikembangkan secara terisolasi pada branch masing-masing sebelum akhirnya digabungkan ke dalam branch utama (main) melalui proses merge yang menangani penyelesaian konflik kode secara sistematis.
